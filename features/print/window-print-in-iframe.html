window.print() inside iframes

Verifies that window.print() opens the print dialog even when it is called from inside an iframe (including a cross-origin iframe).

Expected behaviour: clicking any of the three buttons below opens the browser/app print dialog (or print preview).

Bug behaviour (before fix): on Android the two iframe buttons did nothing — the top-frame button worked, but window.print() from within an iframe was silently ignored. See crbug.com/40342444.
