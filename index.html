Privacy Test Pages

This project contains a collection of pages that are meant to be used for testing various privacy and security features of browsers and browser extensions.

Tracker Reporting

  • 1 major tracker loaded via script
  • 1 major tracker with surrogate
  • 1 major tracker loaded via img
  • Image loaded via document fragment
  • 1 major tracker loaded via fetch

Browser Features

  • Geolocation
  • Links Open in New Window
  • window.close() Link
  • Simple text area for editing
  • URL Schemes

Security

  • Infinite location.reload() loop
  • URL Spoofing - JS page rewrite

Privacy Protections Tests

  • Request blocking
  • Fingerprinting
  • Storage blocking
  • Referrer trimming
  • HTTPS upgrades
