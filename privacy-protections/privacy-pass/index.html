Privacy Pass ACT (draft-schlesinger-privacypass-act) Test

This page tests the browser's Privacy Pass ACT implementation against the test server at http://127.0.0.1:8443.

How it works

Privacy Pass operates at the HTTP layer, not JavaScript. The browser should:

  1. Detect 401 + WWW-Authenticate: PrivateToken from /protected
  2. Run ACT issuance with the issuer (via /token-request)
  3. Spend credits and retry with Authorization: PrivateToken token=<SpendProof>
